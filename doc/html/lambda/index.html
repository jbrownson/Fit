BOOST_FIT_STATIC_LAMBDA

Header

#include <boost/fit/lambda.hpp>

Description

The BOOST_FIT_STATIC_LAMBDA macro allows initializing non-capturing lambdas at compile-time in a constexpr expression.

Example

const constexpr auto add_one = BOOST_FIT_STATIC_LAMBDA(int x)
{
    return x + 1;
};

BOOST_FIT_STATIC_LAMBDA_FUNCTION

Header

#include <boost/fit/lambda.hpp>

Description

The BOOST_FIT_STATIC_LAMBDA_FUNCTION macro allows initializing a global function object that contains non-capturing lambdas. It also ensures that the global function object has a unique address across translation units. This helps prevent possible ODR-violations.

Example

BOOST_FIT_STATIC_LAMBDA_FUNCTION(add_one) = [](int x)
{
    return x + 1;
};
