arg

Header

#include <boost/fit/arg.hpp>

Description

The arg returns a function object that returns the Nth argument passed to it. It actually starts at 1, so it is not the zero-based index of the argument.

Synopsis

template<class IntegralConstant>
constexpr auto arg(IntegralConstant);

template<std::size_t N, class... Ts>
constexpr auto arg_c(Ts&&...);

Example

assert(arg(std::integral_constant<int, 3>())(1,2,3,4,5) == 3);
