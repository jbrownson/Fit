repeat

Header

#include <boost/fit/repeat.hpp>

Description

The repeat function decorator will repeatedly apply a function a given number of times.

Synopsis

template<class IntegralConstant>
constexpr repeat_adaptor<IntegralConstant> repeat(IntegralConstant);

Requirements

IntegralConstant must be:

  • IntegralConstant

Example

struct increment
{
    template<class T>
    constexpr T operator()(T x) const
    {
        return x + 1;
    }
};

constexpr auto increment_by_5 = boost::fit::repeat(std::integral_constant<int, 5>())(increment());
assert(increment_by_5(1) == 6);
