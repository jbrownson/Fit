flip

Header

#include <boost/fit/flip.hpp>

Description

The flip function adaptor swaps the first two parameters.

Synopsis

template<class F>
flip_adaptor<F> flip(F f);

Semantics

assert(flip(f)(x, y, xs...) == f(y, x, xs...));

Requirements

F must be:

  • BinaryCallable
  • MoveConstructible

Example

int r = boost::fit::flip(boost::fit::_ - boost::fit::_)(2, 5);
assert(r == 3);
